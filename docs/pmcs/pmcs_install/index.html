Instalasi

Anda berada di dalam panduan: PMCS

Perhatikan alur proses instalasi yang dibagi sesuai fungsi.

Alur Instalasi

ENTITAS:

  • PROYEK = Tim Proyek
  • M.AKP = Manajer Akuntansi dan Pajak
  • BSI = Biro Sistem Informasi

Bantu saya mengkoreksi kesalahan gambar? Unduh disini.

Panduan Instalasi

Tahap ini cukup dijalankan 1 kali. Pastikan telah memasang daftar aplikasi yang dibutuhkan PMCS.

  1. Siapkan installer aplikasi PMCS, unduh melalui alamat e-accounting.wika.co.id/pmcs.
  2. Klik kanan installer, pilih Run as administrator.

  1. Tunggu proses instalasi hingga selesai. Estimasi hingga 5-10 menit tergantung spesifikasi laptop/komputer.

  1. Diharap tidak menekan tombol Close sebelum tampil tulisan Press any key to continue . . . Tekan tombol Enter setelah tampil tulisan tersebut.

  1. Lanjut, tekan tombol Enter kembali.

  1. Selesai. Tekan tombol Close. Silahkan beralih ke tahap selanjutnya.

Panduan Patch

  1. Unduh patch melalui alamat: e-accounting.wika.co.id/pmcs.
  2. Klik kanan pilih Extract Here.

  1. Buka folder Patch PMCS. Jalankan program, klik kanan pilih Run as administrator.

  1. Lakukan ini apabila terhadang antivirus (opsional).

    Klik tombol More info. Kemudian klik tombol Run anyway.

  1. Klik tombol Lanjut >.

  1. Klik tombol Mulai Patch >.

  1. Tunggu proses instalasi hingga selesai.

  1. Proses selesai, klik tombol Lanjut >.

  1. Selamat! Instalasi patch PMCS telah berhasil dilakukan. Tekan tombol Selesai.

Kode Lisensi

Cara Mendapatkan Kode Lisensi

Apa itu kode lisensi?

Kode lisensi adalah kode unik yang spesifik untuk setiap perangkat laptop/komputer. Oleh karena itu, tidak akan ditemukan kesamaan kode lisensi untuk perangkat yang berbeda meskipun untuk proyek yang sama.

Proses input lisensi hanya dilakukan 1 kali saat awal pengoperasian.

Dimohon mencantumkan data-data berikut pada surat/email permohonan:

  1. Perihal: Permohonan Kode Lisensi PMCS Proyek < nama proyek >
  2. Kode instalasi < XXXX-XXXX > (cara mendapatkannya lihat dibagian Panduan Input Kode Lisensi)
  3. Kode SPK Proyek dari Pemohon < XXXXXX >
  4. Nama Pengguna (spesifik untuk setiap laptop/komputer, dapat berupa NIP)
  5. Unit Kerja atau Departemen Pemohon

Permohonan dengan izin tertulis dapat berupa surat fisik dan atau berbentuk soft copy dikirim melalui e-mail. Lihat daftar kontak.

Panduan Input Kode Lisensi

Proses input kode lisensi dapat dilakukan setelah mendapatkan persetujuan Ketua Tim Implementasi / M.AKP.

Setelah mendapatkan kode lisensi, simpan kode yang Anda dapatkan karena dapat dibutuhkan kembali nanti. Buka Firefox, ketik alamat localhost:9191.

Input kode lisensi pada kolom yang disediakan. Klik tombol Simpan.

Cara Mengetahui Versi PMCS

Buka halaman login, silahkan muat ulang (refresh) browser, tekan tombol F5.

Letak versi terdapat dibawah tombol Login.

Lakukan instalasi patch kembali apabila versi di perangkat Anda bukan versi terkini.
